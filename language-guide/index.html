This interactive guide is an exploration of the possibilities of learning new language system through new mediums.

The language is comprised of an alphabet of modular characters that can be combined in a multitude of configurations to form words, concepts, and phrases. Meant to supplement the user's primary language method, this system was developed with activists and dissident groups in mind as a method for communicating covertly.

This project is a collaboration between Parsons, Design + Technology Graduates, Nour Chamoun and Mala Kumar.
